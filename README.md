# Real-Time Collaboration Editor

A real-time collaborative text editor built using the MERN Stack and Socket.IO.

## Features

- User Registration
- User Login with JWT Authentication
- Protected Routes
- MongoDB Atlas Integration
- Real-Time Text Synchronization
- Socket.IO Communication
- React Frontend
- Express Backend

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Socket.IO
- JWT Authentication
- Axios
